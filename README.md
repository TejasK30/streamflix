# streamkit

- Streamkit is a video transcoding and streaming platform, where users can upload video and then it gets transcoded into video qualities - 360, 480, 720 and 1080p. After uploading of video users can watch this videos.
- HLS streaming is used here.
- Tech Stack:
  - Frontend - Next.js, video.js
  - Backend - Node.js, Express.js, Multer, BullMQ, FFMpeg, Docker, PostgreSQL
